為什麼選擇 PyGMT？

告別繁瑣的 Shell Script，擁抱 Pythonic 的優雅。

Python 優先

完全整合至 Python 生態系。支援 Pandas DataFrame、xarray 與 NumPy 陣列，資料處理與繪圖無縫接軌。

出版級品質

繼承 GMT 的高解析度向量輸出能力。無論是學術期刊還是海報展示，都能產出最精美的地圖。

豐富投影

內建超過 30 種地圖投影法。從麥卡托投影到極地投影，輕鬆處理複雜的地理空間數據。

example_plot.py
import pygmt # 建立繪圖實例 fig = pygmt.Figure() # 繪製海岸線 # region="g" 代表全球 # projection="W15c" 代表 Mollweide 投影，寬度 15cm # frame="a" 自動添加邊框刻度 fig.coast( region="g", projection="W15c", land="darkgray", water="skyblue", frame="a" ) # 顯示圖像 fig.show()

快速安裝

推薦方式 (Conda/Mamba)

這是最穩定的安裝方式，因為 GMT 核心依賴較為複雜。

conda install -c conda-forge pygmt

替代方式 (Pip)

請確保系統中已預先安裝 GMT C 函式庫。

pip install pygmt
