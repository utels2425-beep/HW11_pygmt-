Python 原生體驗

與 Pandas、Xarray 和 NumPy 深度整合。直接將 DataFrame 丟給 PyGMT，無需手動處理繁瑣的數據格式轉換。

出版級畫質

自動處理地圖邊框、刻度與圖例。繼承 GMT 數十年的繪圖美學，輸出高解析度 PDF/EPS，滿足最嚴格的期刊要求。

3D 與投影

支援超過 30 種地圖投影。輕鬆繪製 3D 地形透視圖、等高線疊加，甚至是行星數據（如火星、月球）。

參考自 PyGMT 官方教學影片，涵蓋從基礎製圖到進階 3D 分析的四大主題。

Tutorial by Leo

1. 你的第一張 PyGMT 地圖

學習地圖解剖學。如何設置底圖 (Basemap)、海岸線 (Coastlines)、地圖框 (Frame) 與色標 (Colorbar)。專注於製作資訊豐富且美觀的平面地圖。

  • Map Configuration
  • Layout Design
Tutorial by Max

2. 互動式海洋數據分析

結合 Pandas 與 Xarray。展示如何利用 Python 的 Panel 庫製作「互動式滑桿」，動態觀察不同深度的海洋溫度變化。

  • Interactive Widgets
  • Xarray Integration
Tutorial by Andre

3. 火星奧林帕斯山 3D 視角

探索地球以外的數據。繪製火星最高峰——奧林帕斯山 (Olympus Mons) 的 3D 陰影透視圖，並學習如何添加「概覽嵌入圖 (Inset Map)」來標示位置。

  • Planetary Data
  • 3D Perspective
Tutorial by Wei Ji

4. LiDAR 點雲轉數值地形

處理高密度的 LiDAR 數據（以威靈頓為例）。演示資料預處理流程，將散亂的點雲 (Point Cloud) 轉化為高品質的網格化 (Raster) 地形模型。

  • Point Cloud Processing
  • Grid Generation
