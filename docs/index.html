Interactive Demo: 地形數據繪圖

嘗試載入數據並繪製全球地形圖 (Earth Relief)。

plot_relief.py
import pygmt # 1. 載入地球地形數據 (grid) # resolution="10m" 代表解析度為 10 arc-minutes grid = pygmt.datasets.load_earth_relief(resolution="10m") fig = pygmt.Figure() # 2. 繪製網格影像 (grdimage) # projection="R12c": Winkel Tripel 投影 (寬度12cm) # cmap="geo": 使用地理專用色票 fig.grdimage( grid=grid, projection="R12c", cmap="geo" ) # 3. 添加色標 (Colorbar) # frame 設定刻度、標籤與單位 fig.colorbar(frame=["a2500", "x+lElevation", "y+lm"]) # 4. 顯示圖像 fig.show()
